# Personal Portfolio (Dark Theme)

A clean, developer-style dark portfolio built with HTML, CSS and minimal JavaScript.

## Features
- Dark developer theme
- Responsive layout for mobile & desktop
- Smooth scroll & mobile menu
- Links to GitHub / Projects

## Files

- `style.css`
- `script.js`

Open `index.html` in a browser.
